# SOC Analyst Portfolio

This repository contains hands-on Security Operations Center (SOC) projects focused on:

- Log analysis & security monitoring
- Incident detection and response
- SIEM investigation workflows
- Detection engineering
- Threat hunting fundamentals

## Lab Environment
- Kali Linux (attack simulation)
- Windows Systems + Ubuntu (endpoint log source)
- Wazuh SIEM (log collection and alerting)

## Projects
- SOC Lab Setup
- Incident Response Reports
- Detection Engineering
- Cloud SOC Case Studies
- Threat Hunting Exercises

Each project includes documentation, screenshots, and analysis to demonstrate real SOC workflows.
